# 🌌 Shivalay Portfolio | Powered by RexD

A high-end, modern Cyberpunk-themed portfolio designed for **Shivalay**, featuring sleek dark UI aesthetics, glassmorphism, and advanced scroll-triggered animations.

---

## ✨ Features

-   **Cyberpunk/Neon UI**: A deep-dark theme with vibrant neon cyan and magenta accents.
-   **Fully Responsive**: Optimized for ultra-wide monitors, tablets, and mobile devices.
-   **Advanced Animations**:
    -   **Scroll Reveal**: Smooth entrance animations for sections using `IntersectionObserver`.
    -   **Glitch Effect**: High-end interactive glitch text on the hero section.
    -   **Custom Neon Cursor**: A dynamic, trailing neon cursor (desktop only).
-   **Glassmorphism**: Semi-transparent cards with backdrop blur and glow effects.
-   **Optimized Assets**: Uses **Lucide Icons** for lightweight, scalable vector icons.

## 🛠️ Tech Stack

-   **HTML5 & CSS3** (Vanilla CSS for performance)
-   **JavaScript** (ES6+)
-   **Lucide Icons**
-   **Google Fonts** (Space Grotesk & Inter)
